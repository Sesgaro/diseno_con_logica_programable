En este mini challenge deberan implementar un button debouncer, el proposito de este modulo es filtrar la señar de salida de un boton para hacerlo utilzable para u diseño. Existen debounger electricos (filtro RC) y de software, en nuestro caso se implemente como un modulo Verilog que implementa filtro temporal.

Para esta entrega de mini challenge se espera lo siguiente

1. Pueden usar codigos de otras fuentes, sin embargo, se espera que lo analicen a mucha profundiudad y que entiendan cada uno de los elementos del codigo

2. Que puedan explicar el codigo paso a paso y que conlleva para la simulacion

3, Que entiendan como simular su codigo e interpretar la simulacion

4. Que demuestren fisicamente el funcionamiento del codigo

La idea es que lo presenten directamente al profesor, ya sea Gonzalo o Gilberto, ya sea en persona en o en Zoom
